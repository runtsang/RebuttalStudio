# Stage 4 - Multi Rounds

## What this stage is for

Handle reviewer follow-up questions in later discussion rounds.

## How to use it

1. Paste the follow-up question in the top input box.
2. Write your draft answer in `Draft Editor`.
3. Click `Refine`.
4. The app uses prior discussion context and returns a polished follow-up response.
5. Review and copy the final text back to the reviewer.

## What you should get

- A context-aware, polished follow-up response.
