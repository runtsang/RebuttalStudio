# Stage 5 - Conclusion

## What this stage is for

Create Final Remarks for AC/chairs so they can quickly understand your full rebuttal outcome.

## How to use it

1. Choose a template (`Template`).
2. Fill or edit the summary source on the left (manual mode).
3. Add final rating changes for each reviewer when applicable.
4. Click `Preview` to check final rendering.
5. If you want automatic placeholder filling, click `Auto Fill`, then manually review.

## What you should get

- A structured, easy-to-read final conclusion text.
